# My Portfolio Website

A professional, responsive portfolio website built with pure HTML, CSS, and JavaScript.

## 🌟 Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Modern UI** - Clean and professional design with smooth animations
- **Navigation** - Sticky navigation bar with smooth scroll
- **Hero Section** - Eye-catching introduction section
- **About Me** - Personal introduction and achievements
- **Projects Showcase** - Display your best work with descriptions and tags
- **Skills Section** - Highlight your technical skills
- **Contact Form** - Get in touch with visitors
- **Social Links** - Connect on various platforms
- **Mobile Menu** - Hamburger menu for mobile devices
- **Smooth Animations** - Professional fade-in and hover effects
- **Scroll to Top** - Quick navigation button

## 🚀 Getting Started

### Prerequisites
- A web browser (Chrome, Firefox, Safari, Edge, etc.)
- A text editor (VS Code, Sublime Text, etc.)
- Basic knowledge of HTML, CSS, and JavaScript

### Installation

1. Clone the repository:
```bash
git clone https://github.com/akshathch567-max/my-protfolio.git
cd my-protfolio
```

2. Open the website:
   - Simply open `index.html` in your web browser
   - Or use a live server extension (VS Code Live Server recommended)

## 🎨 Customization

### Update Personal Information

Edit `index.html` and replace:
- Your name in the hero section
- Email address in the contact section
- Phone number and location
- Project descriptions and links
- Skills and achievements

### Modify Colors

Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #ec4899;
    --dark-bg: #0f172a;
    /* ... other colors ... */
}
```

### Add Your Projects

Add new project cards in the Projects section:
```html
<div class="project-card">
    <div class="project-image">
        <i class="fas fa-your-icon"></i>
    </div>
    <h3>Your Project Title</h3>
    <p>Project description</p>
    <div class="project-tags">
        <span class="tag">Tech1</span>
        <span class="tag">Tech2</span>
    </div>
    <a href="your-link" class="btn btn-small">View Project</a>
</div>
```
